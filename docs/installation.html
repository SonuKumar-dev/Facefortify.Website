Installation Guide

Step-by-step instructions to install FaceFortify on your computer

🪟 Windows Installation

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • 500 MB free disk space
  • Camera (webcam or built-in)
  • Administrator rights (for first-time setup)

Installation Steps

1 Download the Installer

Download FaceFortify-Setup-1.0.0.exe from the download page.

💡 Note: Your browser may warn about the download. Click "Keep" - this is normal for new software that isn't yet widely distributed.
2 Run the Installer

Double-click FaceFortify-Setup-1.0.0.exe to start installation.

⚠️ Windows Defender Warning: If Windows shows "Windows protected your PC", click "More info" → "Run anyway". This happens because the app isn't code-signed yet.
3 Follow Setup Wizard
  1. Click "Next" on the welcome screen
  2. Choose installation folder (or keep default)
  3. Select if you want a desktop shortcut
  4. Click "Install"
  5. Wait for installation to complete (~30 seconds)
4 Launch FaceFortify

After installation completes:

  • Check "Launch FaceFortify" and click "Finish", OR
  • Find it in Start Menu → Search "FaceFortify"
✅ Installation Complete! FaceFortify will ask for camera permission on first launch.

Uninstallation

To remove FaceFortify:

  1. Go to Settings → Apps → Installed apps
  2. Find "FaceFortify" in the list
  3. Click the three dots → Uninstall
  4. Follow the prompts
💡 Your Data: Your encrypted folders and face data are stored separately in %APPDATA%\FaceFortify. They won't be deleted when you uninstall. Delete this folder manually if you want to remove all data.

🐧 Linux Installation

System Requirements

  • Linux kernel 4.0 or later (x86_64)
  • 500 MB free disk space
  • Camera with V4L2 support
  • FUSE (for AppImage) - usually pre-installed

Supported Distributions

  • Ubuntu 20.04 and later
  • Fedora 35 and later
  • Debian 11 and later
  • Arch Linux
  • Any modern distribution with kernel 4.0+

Installation Steps

1 Download AppImage

Download FaceFortify-Linux-x86_64.AppImage from the download page.

2 Make it Executable

Open Terminal in your Downloads folder and run:

chmod +x FaceFortify-Linux-x86_64.AppImage

Or using File Manager:

  1. Right-click the AppImage file
  2. Select "Properties"
  3. Go to "Permissions" tab
  4. Check "Allow executing file as program"
3 Run FaceFortify

Double-click the AppImage file, or run in terminal:

./FaceFortify-Linux-x86_64.AppImage
✅ That's it! FaceFortify will launch. No installation needed - it's portable!

Optional: Install to System

To add FaceFortify to your application menu:

  1. Move the AppImage to a permanent location:
    mkdir -p ~/Applications
    mv FaceFortify-Linux-x86_64.AppImage ~/Applications/
  2. Run it once - it will offer to integrate with your desktop
  3. Click "Yes" to add to application menu

Camera Permissions

💡 Camera Access: On first launch, your system may ask for camera permission. Click "Allow". If the camera doesn't work, check your privacy settings or run: sudo chmod 666 /dev/video0

Uninstallation

Simply delete the AppImage file. Your data is stored separately in ~/.local/share/FaceFortify.

🍎 macOS Installation

System Requirements

  • macOS 10.15 (Catalina) or later
  • 500 MB free disk space
  • Built-in camera or external webcam

Choose Your Version

  • Intel Macs: Download FaceFortify-macOS-x64.dmg
  • Apple Silicon (M1/M2/M3): Download FaceFortify-macOS-arm64.dmg
💡 Not sure? Click Apple menu () → About This Mac. If it says "Apple M1/M2/M3", download arm64. Otherwise, download x64.

Installation Steps

1 Download and Open DMG

Download the appropriate DMG file and double-click to open it.

2 Drag to Applications

A window will open showing the FaceFortify app and your Applications folder. Drag FaceFortify to Applications.

3 First Launch (Important!)

Go to Applications folder and find FaceFortify.

⚠️ Security Warning: macOS will say "FaceFortify cannot be opened because the developer cannot be verified." This is normal for unsigned apps. Follow these steps:
  1. Right-click (or Control+click) the FaceFortify app
  2. Select "Open" from the menu
  3. Click "Open" in the warning dialog
💡 Note: You only need to do this once. After the first time, you can open it normally.
4 Grant Camera Permission

When FaceFortify asks for camera access:

  1. Click "OK" in the permission dialog
  2. If you accidentally denied it, go to:
    System Preferences → Security & Privacy → Camera
    Check the box next to FaceFortify
✅ All Set! FaceFortify is now installed and ready to use.

Alternative Method: Terminal

If the right-click method doesn't work, you can use Terminal:

xattr -cr /Applications/FaceFortify.app

Then open the app normally.

Uninstallation

  1. Open Applications folder
  2. Drag FaceFortify to Trash
  3. Empty Trash
💡 Your Data: Your encrypted folders and settings are stored in ~/Library/Application Support/FaceFortify. Delete this folder manually if you want to remove all data.

🚀 After Installation

First Time Setup

  1. Register Your Face
    • Click "Get Started" or "Register Your Face"
    • Allow camera access when prompted
    • Look at the camera - it will capture 30 frames
    • Keep your face in the green box
    • Takes about 30 seconds
  2. Set Backup Password (Recommended)
    • In case you change computers or hardware
    • Store it in a password manager
    • You can skip this, but it's safer to set one
  3. Add Folders to Lock
    • Click "Add Folder" on the main screen
    • Select folders you want to protect
    • Click "Lock All" to encrypt them
🎉 You're Ready! Your folders are now protected with facial recognition and AES-256 encryption.

🔧 Troubleshooting

Camera Not Working

  • Windows: Settings → Privacy → Camera → Allow apps to access camera
  • Linux: Check if camera works in other apps. Try: ls /dev/video*
  • macOS: System Preferences → Security & Privacy → Camera → Enable FaceFortify

App Won't Launch

  • Windows: Right-click → Run as Administrator
  • Linux: Make sure AppImage is executable: chmod +x FaceFortify*.AppImage
  • macOS: Use Right-click → Open method (see installation steps)

Face Not Recognized

  • Ensure good lighting (not too dark or bright)
  • Face the camera directly
  • Remove glasses temporarily during registration
  • Keep face in the green detection box
