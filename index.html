Security Stack Analysis

Inspect the core algorithms securing your folders offline

ArcFace Neural Network

FaceFortify integrates the state-of-the-art ArcFace AI model directly into your local machine. The model maps 30 separate coordinates to construct a secure, 512-dimensional vector signature.

  • Accuracy metrics score 99%+ on standard benchmarks
  • Validation processes 10 verification frames in under 1 second
  • Live eye-tracking prevents photo-based spoofing attempts

PBKDF2 Key Derivation

Encrypted key files are protected from brute force attacks using PBKDF2 hash stretching (utilizing HMAC-SHA256). The process stretches input vectors over 100,000 CPU iterations.

  • Derives keys using a dynamically generated unique cryptographed salt
  • Slows dictionary attack attempts down to 10 guesses per second
  • AES-256-CBC encrypts individual files sequentially

Machine Hardware Binding

To guarantee that stolen hard drives or copied keyfiles remain completely useless on other computers, FaceFortify derives hashes directly from machine hardware signatures.

  • CPU serial numbers and motherboard serial identifiers are queried
  • Primary network interfaces' hardware MAC addresses are mixed in
  • Backup passwords can recover data if hardware needs repairs

OS-Level Permissions Lock

Besides binary file encryption, the desktop app modifies system access control permissions dynamically on folders.

  • Applies OS Access Control Lists (ACL) on Windows hosts
  • Fires system chmod updates directly on Linux and macOS devices
  • Restricts other local user profiles from accessing locked paths
